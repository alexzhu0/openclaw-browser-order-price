# Release 2026.03.24

## 概要

第三次升级，重点解决长批次调试、多账号并发和结果分类问题。

## 主要内容

- `verbose=true` 时自动把运行日志写入文件
- 新增 `checkout_blocked` 状态：
  - 点击购买后只拿到商品页 fallback 价格
  - 点击购买后始终没有进入 checkout
- 新增 `checkout_blocked` 自动重试配置
- 增加单机多账号并发能力
- 新增 `accountA/accountB` 目录式管理输出、状态、日志和生成配置
- 多账号结果支持合并导出 JSON 和 Excel
- `prepare-login` 支持 `--profile-dir`，便于独立账号窗口隔离
- README 和 AGENTS 同步更新到多账号和 Excel 导出版本

## 实测记录

- 使用全量 `test.json` 做批量极限测试
- `behavior-1` 可以稳定跑完整批次，但中途仍可能遇到登录恢复
- `behavior-2` 速度更快，双账号并发时整体表现健康
- `behavior-3` 继续拉大等待参数后，`accountA` 在实测中稳定性更高

## 结论

- 这版开始，项目已经具备“多账号真实跑批 + 留证据 + 事后分析”的基本能力
